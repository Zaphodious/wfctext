Wave Form Collapse, Roguelike Edition

This is a demonstration of the Wave Form Collapse algo (in tileset mode), implimented for text in js. Each example is fed from a txt file containing a tileset, and is 12 by 12 tiles (34 by 34 characters). Maps do not have to be square, I just like them that way.

Please wait while the examples generate. Refresh to see new examples.

The code is available at https://github.com/Zaphodious/wfctext
